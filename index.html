קוד PPTX ל-RTL והעלאה ל-GitHub

קוד Python להמרת טקסט ב-PPTX

להלן דוגמה לקוד בפייתון המשתמש בספריית python-pptx לטעינת קובץ PPTX, מעבר על כל השקופיות והצורות המכילות טקסט, והגדרת כיוון הטקסט לימין-שמאל.

from pptx import Presentation

# קובץ המקור וקובץ היעד
input_pptx = 'original.pptx'
output_pptx = 'rtl_presentation.pptx'

# טעינת המצגת
prs = Presentation(input_pptx)

# מעבר על כל השקופיות והצורות עם טקסט
for slide in prs.slides:
    for shape in slide.shapes:
        if shape.has_text_frame:
            for paragraph in shape.text_frame.paragraphs:
                # הוספת הגדרה לכיוון טקסט לימין לשמאל (בידי)
                pPr = paragraph._p.get_or_add_pPr()
                pPr.set('bidi', '1')

# שמירת המצגת החדשה
prs.save(output_pptx)
print("המצגת נשמרה כ-", output_pptx)

העלאת הקוד ל-GitHub

כדי להעלות את הקוד ל-GitHub, ניתן לבצע את הפעולות הבאות בטרמינל:

  1. נווט לתיקיית הפרויקט והפעל את הפקודה: 
    git init
  2. הוסף את כל הקבצים למאגר: 
    git add .
  3. בצע commit עם הודעה: 
    git commit -m "Initial commit - הוספת קוד PPTX ל-RTL"
  4. הגדר את המאגר המרוחק (החלף username ו-repository בפרטי המאגר שלך): 
    git remote add origin https://github.com/username/repository.git
  5. העלה את הקוד למאגר: 
    git push -u origin master

    אם המאגר שלך משתמש ב-branch בשם main, יש להשתמש ב-git push -u origin main.
